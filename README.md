## IMC Calculator


![preview](/github/preview.gif)


> In this project I created a BMI (IMC) calculator. It calculates the imc according to the data entered by the user and classifies it according to the WHO (OMS) table.

> This project was done as an exercise proposed by Rocketseat's Explorer course.
